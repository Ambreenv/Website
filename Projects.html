PROJECTS

Website

During my second co-op, I began making a website to apply and implement the HTML&CSS skills that I taught myself. Being able to create a website from scratch and designing the UX/UI allowed me to better understand and appreciate the process that goes into making a website. I created this site in hopes to improve my coding skills and learn how to turn my ideas and designs a reality.

When I first decided to make this webiste, I wasn't sure where to begin or how to start. I began watching tutorial videos on Youtube and readings posts on Stack Overflow to help me get started and navigate my way through the process of creating a website. The tutorial videos that I watched helped me build the general outline of the website, such as the header and the navigation bar. As I began to understand the langauge more, I was able to figure out the rest on my own to create the website that I had in mind.

I really enjoyed building my website as it was a lot of fun to come up with the layout, color schemes, and design as well as tap into my creative side. The project from start to end took me about 2-3 weeks to build and publish, however I had the bulk of it done in 3 days since I had a lot of fun working on it during my free time. After the majoity of the website was done, the rest of the time was spent ironing out the kinks and trying to polish the design to the best of my ability.

This project allowed me to learn a lot more about HTML & CSS, and realize the amount of detail, time, and effort that goes into a website and how hard it can be to get it the way you want. I recommend that anyone who has ever wanted to create a website should go and create one because you will feel so accomplished in the end and learn way more than you thought you would!


Don't Be Alarmed

In the summer of 2017, during my first co-op position, I designed a Printed Circuit Board (PCB) to create a pressure sensor alarm. Components such as transistors, diodes, resistors and timers were soldered onto the board allowing the pressure alarm to output a high pitch sound. When I first attempted this project my soldering skills were horrendus, but after soldering a lot during the summer I was practially a pro by the end (okay, not really) but I did improve a lot.

By completing this project I was able to figure out why each component was necessary in order to create the alarm. I was also able to better understand what each individual component does and how they all worked together so that the alarm would be able to properly function. Although the project was on the smaller side, it was fulfilling to see the final product and its completion from start to end.



The PongBot

To conclude my 1A term of Mechatronics Engineering, I got to create an autonomous robot for my end-of-term project which was able to partcipate in a game of Beer Pong. The final project allowed for a lot of creativity as it was very open-ended since the students could essentially create whatever they desired, as long as it fit the guidelines that were provided. My group and I decided to create an autonomous beer pong playing robot, called the PongBot, which challenged contestants in a game of beer pong. My group and I (pictured on the right) figured it would be a fun project to make that could also help us win at parties!

The user got to select the cup they wanted to aim at and the robot was able to align itself to the targeted cup, and fire the ball using the launching mechanism that was created. The robot that was created was able to scan its side for cups to see if the game was still going on, or if the opponent had won. Whenever the robot sunk a ball, the user pressed a button on the robot which kept a counter of the number of balls sunk. To make the robot as autonomous as possible, color, touch and ultrasonic sensors were implemented so that the user interaction was minimal and the robot was able to align itself and launch the ball autonomously.

Throughout this project many of the topics that were taught to me throughout the term, as well as skills that I had developed throughout high school and university, were able to be applied during this project. I even got to go to a machine shop to design the track that robot would be placed on so that it could scan for the cups (pictured below).

Overall this project was really fun to create, and it allowed me to implement and improve my coding, problem solving, and design skills to create a project that I am very proud of.
